How it works

  1. Paste your post — plain text or text you've already styled elsewhere. Emojis stay as they are.
  2. Select what to style — highlight words, or tap line numbers on the left (shift-tap for multiple lines).
  3. Pick a style — the font list appears when something is selected. Tap a style to apply it.
  4. Copy & post — hit Copy styled post, then paste straight into X.

Hashtags and @mentions can stay normal using the toggles. The character counter is an estimate.
